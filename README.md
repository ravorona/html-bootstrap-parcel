Bootstrap blank HTML app with [Parcel](https://parceljs.org) bundler

## Commands
### Lint assets
```bash
# Lint stylesheets & javascripts
yarn lint

# Lint stylesheets only
yarn lint:css

# Lint javascripts only
yarn lint:js
```
### Build assets
```bash
# Development mode
yarn build

# Production mode
yarn build:production

# Hot reload mode
yarn hot
```
